About : visualization

  • Structure

    • Structures are provided by the scenes. Presentation is enhanced by providing omnipresent navigation controls.

  • Scene Utilization

    • Users can navigate the scenes through the controls provided on each page. Consistent look and feel retains the user focus on the content. Simple html navigation was used to acheive this.

  • Annotations

    • Tooltip and label based annotations have been provided on all the screens, as applicable

  • Parameters

    • Dropdown on scatter plot page is one of the parameters used in the project. The navigation also carries state parameters in certain cases.

  • Triggers

    • Bubble chart has a click trigger to show detailed data on scatterplot. Scatterplot page has a dropdown which triggers filtering by Neighborhood.
