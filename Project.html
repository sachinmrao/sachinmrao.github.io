NYC AirBnB data analysis with d3.js

Modern platforms like AirBnB are unique, as they provide detailed reviews and are budget friendly.

This is an attempt to help customers gain insight into AirBnB NYC

  • Plot ratings vs price, to find the best value
  • Ability to filter by location

‘Airbnb Listings in New York City’ is a publicly available dataset hosted on the Tableau resources page. This csv was generated from an xlsx file contains 30,478 Airbnb listings in New York City. This data was compiled by Airbnb. Data was sanitized by removing NULL values. The following filters were applied to obtain the desired subset:

  • Property Type: Apartment
  • Listing Type: Entire home/apt
  • Review Scores Rating is not null
